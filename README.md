### 📋 Sobre o Projeto
Este projeto consiste em um sistema de gestão para cozinhas de restaurantes, permitindo o controle de insumos, pratos e equipe. Foi desenvolvido com o objetivo de centralizar as operações do dia a dia de uma cozinha profissional, desde o estoque de ingredientes até o cardápio disponível.

### ✨ Funcionalidades Implementadas
- **Modelos de Dados:** Estrutura completa para `Ingrediente`, `Prato` e `Cozinheiro`.
- **Listagem e Gerenciamento:** CRUD completo (Criação, Leitura, Atualização e Exclusão) para gerenciar o estoque e o menu.
- **Funcionalidade de Busca:** Filtro inteligente para encontrar pratos ou ingredientes de forma rápida.
- **Design Responsivo:** Interface moderna utilizando **Bootstrap**, adaptável a dispositivos móveis e desktops.

---

### 🚀 Tecnologias Utilizadas
- **Python** (Lógica de back-end)
- **Django** (Framework Web)
- **HTML5 / CSS3** (Estrutura e estilo)
- **Bootstrap** (Framework de interface e responsividade)
- **SQLite** (Banco de dados padrão do Django para desenvolvimento)

### 💻 Como rodar o projeto?

Siga os passos abaixo para configurar o ambiente em sua máquina local:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/SEU-USUARIO/cozinha-restaurante.git
   cd cozinha-restaurante
   ```

2. **Crie um ambiente virtual:**
   ```bash
   python -m venv venv
   ```

3. **Ative o ambiente virtual:**
   - **Windows:** `venv\Scripts\activate`
   - **Linux/Mac:** `source venv/bin/activate`

4. **Instale as dependências:**
   ```bash
   pip install django
   # Ou se tiver um requirements.txt:
   # pip install -r requirements.txt
   ```

5. **Execute as migrações do banco de dados:**
   ```bash
   python manage.py migrate
   ```

6. **Inicie o servidor de desenvolvimento:**
   ```bash
   python manage.py runserver
   ```

7. **Acesse no navegador:**
   Abra [http://127.0.0.1:8000](http://127.0.0.1:8000)
